# CellScope

Browser-based particle size measurement tool for TEM microscope images.

## Features

- Load TIF/TIFF/PNG/JPG microscope images
- Scale bar calibration (nm/μm)
- Manual particle measurement (major & minor axis)
- Auto-computed: aspect ratio, equivalent diameter
- Statistics: mean, std dev, scatter plot
- Undo/redo support
- CSV & annotated image export
- Dark theme UI
- Magnifier for precision annotation
- Drag-to-modify endpoints
- Minimap navigation
- LocalStorage persistence

## Usage

Open `index.html` in a browser, or visit the [GitHub Pages deployment](https://m2kar.github.io/CellScope/).

No build step required — pure vanilla JS.
